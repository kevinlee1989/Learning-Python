1. 전역(global) 변수와 지역(local) 변수
    (1) argument(논쟁이라는 뜻도 있지만 인수), value(값), parameter(전달값)
        Beatles(변수) = 50
        test(Beatles)   # 파이썬 입장 Test(50) 50이라는 값을 가지고 태운다
                        # def Test(BTS): # 주시는 값을 저는 BTS라고 명칭하도록하겠슴다
                            print(BTS)

        Beatles = 100
        Test(Beatles)     # Test(100) 이 출발한다
                          # def Test(Beatles):
                               print(Beatles)
                               Beatles = 200

        print(Beatles) => 100

        (2) 전역변수와 지역변수의 차이점
            프로그램에서 변수를 사용할 수 있는 범위가 존재
            전역변수 : global variables
            지역변수 : local이라고 하며 함수 내에서만 사용가능 (if, while, for문에서 선언된 변수는 global)

            전역변수 : 함수 내에서 시작하지 않은 모든 변수
            지역변수 : 함수 내에서 시작한 변수

            *** 함수 내에서 사용된 변수는 함수를 나가는 순간 소멸
            *** 함수 내에서 전역변수와 동일한 이름을 쓰더라도 지역변수이다 이멍청한놈아.


2. 표준모듈 time
    import time

    time.time() : 현재시간. 1970년 1월 1일 0시0분0초를 기준으로 지금 몇초냐
    time.ctime(시간) : 인수로 전달된 시간을 형식을 갖춰서 반환
    time.strftime() : 형식을 갖춘 데이터 생성, '%Y%m%d %H:%M:%S'
    time.sleep(정수): 입력한 초만큼 중지

    (2) 파일명

        open('C:/sample.txt') # 절대경로이다  # \는 특수기호이기때문이다 \를 쓰는순간 다음글자가 뭔지를 궁금하게된다

    (3) 옵션
        r : 읽기모드, 없으면 이거(default)
        w : 쓰기모드, 파일이 없으면 만듬, 있으면 덮어씀 #별로 안씀
        a : 쓰기모드, 파일이 없으면 만듬, 있으면 맨 뒤에 이어서 씀
        x : 쓰기모드, 파일이 없으면 만듬, 있으면 오류

        t : 텍스트모드, 우리 눈에 읽힘
        b : 바이너리모드, 음성, 이미지, 영상
